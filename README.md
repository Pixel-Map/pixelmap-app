# pixelmap-app

This is the repository for the Pixel Map App. It is a react-native app based on expo. 

## Installation

The first time one needs to run:

```
yarn install
```

For running it on iOS, one needs first to run:
```
cd ios && pod install
```



## Running the app

```
expo start
```

or

```
yarn android
```

or

```
yarn ios
```
